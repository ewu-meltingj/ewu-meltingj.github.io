My name is Jeremy

Hello Everybody my name is Jeremy Meltingtallow. I am a senior(YEA!!) at Eastern Washington University working on my Degrees for Computer Information Systems and Visual Communication Design.

Lets get deeper...

My first drawing that I can remember was a shoe made out of chalk on my driveway. The arches were magnificent and I think a couple cats sacrificed their bodies to color the Dorthy red leather. After my art won me a key to the city I moved to smaller things. Now I mainly work with vector graphics and write server applications.

Background Information

I started using HTML in late 1999 and CSS in 2005. After CSS I began learning Javascript, C, Java and a smidgen of ruby. I host a blog using a custom CMS and Server both of course written by myself using Nodejs :).
