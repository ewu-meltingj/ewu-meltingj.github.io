HELLOARYBODY

MY NAME IS Jeremy Meltingtallow

I am a 27yr old human being with a passion for design. I have been active in the web development scene for over 14 years. From tables to floats and from trivial page loads to asyncronous awesomenetss, I have seen some things.

Design Philosophy

Focus on a clean interface where the user needs not to think. Color has its place but one should defer from a loud design if it takes away from a transparent user experience. Basically I design as if the user is drunk.

Tools of Choice

My current setup include Sublime Text as my text editor. For version control I use github. When utilizing git I switch between the OSX app and the command line on my laptop. Beyond editing my other favorite tools include Nodejs, npm, Angularjs, Cordova, and jekyll. This excludes countless other tools. Thus these are tools that I am currently using on mobile/desktop projects.
